Mouse Events

Variants

MouseMove

LButtonDown

Indicates that the left mouse button is pressed.

RButtonDown

MButtonDown

LButtonUp

Indicates that left mouse button is released.

RButtonUp

MButtonUp

Indicates that middle mouse button is released.

LButtonClick

RButtonClick

MButtonClick

MouseWheel

Positive/negative means forward/backward scrolling.

MouseHWheel
